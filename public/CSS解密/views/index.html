切角效果实现：

由渐变、background-size组合实现

切角效果实现由CSS渐变、background-size组合实现各种。
切一个角
background: linear-gradient(-45deg,transparent 15px,burlywood 0);

切角效果实现由CSS渐变、background-size组合实现各种。
切两个角,分左右背景，填充颜色，通过设置角度缺省
background: linear-gradient(-45deg,transparent 15px,burlywood 0) right, linear-gradient(45deg,transparent 15px,burlywood 0) left; background-size: 50% 100%; background-repeat: no-repeat;

(弧形)切角效果实现改为CSS径向渐变，其他设置保持一致。
四个角都是弧形：
background: radial-gradient(circle at top left,transparent 15px,burlywood 0) top left;

以上实现缺点在于代码冗余、无法使用动画效果
由内联的SVG、border-image实现:
background-clip: padding-box; border:15px solid burlywood; border-image: 1 url('data:image/svg+xml,');

上面的方案局限在于无法设置渐变、纹理背景等。
使用`clip-path`剪切,元素背景可设置图片、渐变等效果：
clip-path: polygon( 20px 0,calc(100% - 20px) 0, 100% 20px, 100% calc(100% - 20px),calc(100% - 20px) 100%, 20px 100%,0 calc(100% - 20px),0 20px );
