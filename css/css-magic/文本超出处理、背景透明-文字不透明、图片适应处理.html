  1. 文本超出省略，显示... 基本要素，有固定的宽度，文本会超出；设置文本超出后不换行；

    .box{
	width:100px;
  	height:30px;
  	background:#000;
  	color:#fff;
  	line-height:30px;
	}
.box p{	
	white-space:nowrap;          // 或 white-space:pre;
	overflow:hidden;
	text-overflow:ellipsis;
}
